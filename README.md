# 📱 Calculadora de IMC (Flutter)

Aplicativo mobile desenvolvido em Flutter para cálculo de IMC com foco em experiência do usuário, validação de dados e animações.

## ✨ Funcionalidades

- Splash screen com animação
- Validação de entrada de dados (altura e peso)
- Cálculo automático do IMC
- Exibição do resultado com gauge animado
- Classificação corporal (OMS)
- Feedback visual com mensagens de erro
- Navegação entre telas

## 🧠 Conceitos aplicados

- Programação orientada a objetos
- Separação de responsabilidades (Service Layer)
- Gerenciamento de estado com StatefulWidget
- Tratamento de exceções
- UX com animações e transições

## 🚀 Tecnologias

- Flutter
- Dart
- Syncfusion Flutter Gauges

## 📌 Objetivo

Projeto desenvolvido para prática de desenvolvimento mobile com foco em boas práticas, organização de código e experiência do usuário.
